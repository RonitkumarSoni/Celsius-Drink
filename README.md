## 📖 About The Project

Welcome to the **Celsius Drink Interactive Landing Page**! This repository holds the source code for a high-performance web experience engineered to provide a seamless user journey while highlighting technical proficiency in front-end development, professional animations, and modern UI/UX principles. Built with a strict focus on clean architecture, performance, and aesthetic design.

Whether you're a recruiter, fellow developer, or just curious about modern web animations, feel free to explore!

**Live Demo:** [https://RonitkumarSoni.github.io/Celsius-Drink/](https://RonitkumarSoni.github.io/Celsius-Drink/)

---

## ✨ Highlights & Features

- ⚡ **Blazing Fast Performance:** Powered by [Vite](https://vitejs.dev/) for optimal build times and snappy client-side rendering.
- 🎨 **Sleek & Responsive Design:** Crafted with [Tailwind CSS](https://tailwindcss.com/) to ensure it looks gorgeous on any device—from mobile screens to ultrawide desktop monitors.
- 🧩 **Modular Architecture:** Built using functional React components for maximum reusability and clean codebase management.
- 🚀 **Interactive Elements:** Engaging micro-animations, cinematic GSAP scroll effects, pinned sections, and fluid clip-path reveals for a highly dynamic browsing experience.
- 📱 **Mobile-First Approach:** Fully optimized for seamless viewing on smaller screens without compromising on features or visual fidelity.
- 🎬 **Scroll-Choreographed Storytelling:** Every section is precisely timed to the user's scroll progress, offering a premium editorial feel.

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites
Make sure you have Node.js (v18+) and npm installed.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/RonitkumarSoni/Celsius-Drink.git
   ```

2. **Navigate into the directory:**
   ```bash
   cd Celsius-Drink
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Build for production:**
   ```bash
   npm run build
   ```
